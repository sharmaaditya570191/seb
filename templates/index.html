The button you see below is a room with a randomly generated name. Click to enter it.

If you enter as a guest, the program will look for all available staff, randomly pick one, and open the room in his/her browser.

If you are staff, you need to open the first room you see to indicate to the program that you are ready.

The program limits staff to 3 chats each.

Clicking on the button with the name of your chat window once opens it. Clicking it a second time closes it, and also deletes the room from the database.

This is a very new program. There are still a lot of bugs.
